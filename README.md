# 🧠 AxonOS – 32-bit x86 operating system from scratch

[![GitHub last commit](https://img.shields.io/github/last-commit/Paullknya/AxonOS?style=flat-square)](https://github.com/Paullknya/AxonOS)
[![GitHub code size in bytes](https://img.shields.io/github/languages/code-size/Paullknya/AxonOS?style=flat-square)](https://github.com/Paullknya/AxonOS)
[![License](https://img.shields.io/badge/license-MIT-blue?style=flat-square)](LICENSE)

> **AxonOS** – a hobby operating system named after the **axon**, the part of a neuron that carries signals.  
> Just as an axon transmits impulses, the kernel connects hardware and software.  
> Written in assembly (NASM) and C. Runs in protected mode, supports a command line, system calls, rings 0/3, and boots from FAT12.  
> **Planned:** FAT32, multitasking, paging, and 64‑bit long mode.
